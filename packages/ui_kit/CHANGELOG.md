## 0.0.1

* Initial release of the UI Kit package.
* Implemented core theming system (`AppTheme`, `AppColors`, `AppTypography`).
* Added layout components:
    * `ResizableSplit`: Horizontal and vertical split views with draggable handles.
    * `Panel`: Standardized container with header, body, and footer.
    * `Tabs`: Custom tab bar and view switching.
* Added overlay components:
    * `CommandPalette`: Modal for searching and executing commands.
    * `ToastOverlay`: Stackable, auto-dismissing notifications.
* Added data display components:
    * `SuperListView`: High-performance list wrapper.
    * `ListItem`: Standardized list rows.
    * `Avatar` and `Badge`.
* Added input components:
    * `Button`, `IconButton`.
    * `TextInput`, `ComposerInput`.
* Added app-specific components:
    * `SettingsHost` for structured settings pages.
    * `DebugPanel` for runtime inspection.
